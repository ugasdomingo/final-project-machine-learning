3. Documentación de la API

Los endpoints REST están documentados automáticamente en /docs. Cualquier sistema externo puede integrar directamente contra esos endpoints usando el header X-API-Key. Para integraciones sencillas usa POST /predictions/simple; para PMS con todos los datos, POST /predictions.
